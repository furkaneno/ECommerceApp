# 🛍 Lumière Shop — Fullstack E-Commerce Application

A modern, full-featured e-commerce web application built with **ASP.NET Core MVC (.NET 8)**, designed to simulate a real-world online shopping experience.

---

## 🚀 Features

### 👤 Authentication

* User Registration & Login
* Secure Password Hashing
* Session-based Authentication

### 🛒 Shopping Experience

* Browse products by category
* Product detail pages
* Search & filtering
* Add to cart / remove / update quantity

### 📦 Orders

* Checkout system
* Order creation
* Order history & details

### 👑 Admin Panel

* Add / Edit / Delete products
* Manage categories
* Full product control

---

## 🧱 Tech Stack

* **.NET 8**
* **ASP.NET Core MVC**
* **Entity Framework Core (Code First)**
* **SQL Server**
* **Razor Views**
* **Bootstrap 5**

---

## 🗄 Database

The application uses SQL Server with automatic database creation.

```json
"Server=.;Database=ECommerceAppDb;Trusted_Connection=True;TrustServerCertificate=True;"
```

✔ Database is created automatically on first run
✔ No manual migration needed

---

## ⚙️ How to Run

1. Open the solution in **Visual Studio**
2. Press **F5**
3. The application will:

   * Create the database automatically
   * Seed initial data
   * Launch in browser
